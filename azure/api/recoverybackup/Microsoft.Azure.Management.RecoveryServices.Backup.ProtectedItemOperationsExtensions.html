  • CreateOrUpdateProtectedItem(IProtectedItemOperations, String, String, String, String, String, ProtectedItemCreateOrUpdateRequest, CustomRequestHeaders) CreateOrUpdateProtectedItem(IProtectedItemOperations, String, String, String, String, String, ProtectedItemCreateOrUpdateRequest, CustomRequestHeaders)

    Summary

    The Create Or Update Protected Item Operation is used to enable protection of an item within a container or to modify the protection of an already protected item. In the case of enable protection, this operation creates a new protected item entity in the backend service. In the case of modify protection, this operation updates the already created entity in the backend service. This is an asynchronous operation. To determine whether the backend service has finished processing the request, call the Get Item Operation Result API.
