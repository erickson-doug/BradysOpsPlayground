  • BeginRefreshAsync(String, String, CustomRequestHeaders, String, CancellationToken) BeginRefreshAsync(String, String, CustomRequestHeaders, String, CancellationToken)

    Summary

    The Begin Refresh Operation triggers an operation in the service which would discover all the containers in the subscription that are ready to be protected by your Recovery Services Vault. This is an asynchronous operation. To determine whether the backend service has finished processing the request, call Get Refresh Operation Result APIs.
