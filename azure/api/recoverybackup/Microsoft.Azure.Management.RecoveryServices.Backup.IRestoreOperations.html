Introduction

The Resource Manager API includes operations for triggering and managing restore actions of the items protected by your Recovery Services Vault.

Methods

  • TriggerRestoreAsync(String, String, CustomRequestHeaders, String, String, String, String, TriggerRestoreRequest, CancellationToken) TriggerRestoreAsync(String, String, CustomRequestHeaders, String, String, String, String, TriggerRestoreRequest, CancellationToken)

    Summary

    The Trigger Restore Operation starts an operation in the service which triggers the restore of the specified item in the specified container in your Recovery Services Vault based on the specified recovery point ID. This is an asynchronous operation. To determine whether the backend service has finished processing the request, call Get Protected Item Operation Result API.
