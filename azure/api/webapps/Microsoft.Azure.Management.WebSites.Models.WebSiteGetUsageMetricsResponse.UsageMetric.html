  • Limit Limit

    Summary

    Optional. The maximum point that the given metric can reach. The limit resets at the time specified by the NextResetTime element. If the limit is reached before the reset time, the site will be stopped. A value of -1 means unlimited. Resources with limits typically include CPU Time, Memory Usage, and File System Storage. Important: Site level Limits information is only meaningful for Shared and Standard web sites. For free web sites, limits are enforced at the webspace level.
